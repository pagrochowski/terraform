Visual Basic Macro Example

Here's a simple Visual Basic macro for Excel:


Sub FormatWorksheet()

' Get the active worksheet
Dim ws As Worksheet
Set ws = ActiveSheet

' Format cells A1:C10
With ws.Range("A1:C10")
    .Font.Bold = True         ' Make text bold
    .Interior.Color = vbYellow ' Fill with yellow
    .HorizontalAlignment = xlCenter ' Center align text
End With

End Sub
